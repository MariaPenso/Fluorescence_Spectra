# Fluorescence_Spectra
A reference library of Excitation-Emission Matrix (EEM) fluorescence spectra collected from everyday consumer packaging and plastic products. 
Spectra were acquired using the Horiba FluoroMax Plus bench-top spectrofluorometer at the University of Minho.
This repository is intended as an open-access spectral archive for researchers, educators, and practitioners working with plastic identification and characterisation.
